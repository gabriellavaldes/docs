**For platform engineers,** Aptible provides customizable infrastructure with a range of tools and settings that allow platform engineers to design and implement infrastructure requirements in a straightforward and manageable way.

Aptible features platform engs ❤️:

- [Managed Databases]() and 24x7 urgent support
- Security and reliability best practices out-of-the-box
- The Security & Compliance dashboard - making HIPAA, HIRUST, and SOC2 a breeze
- Granular-user permissions


## Why developers love Aptible
**For developers,** the appeal of Aptible lies in its simplicity and developer-friendly features.

 It’s **simple** like Heroku, but way **more powerful**.

Aptible features developers ❤️:

- Deploy code in seconds, with zero downtime
- Blue-green deployments
- Instant rollbacks for failed deployments and high-availability app deployments — by default
- Robust [CLI]() and developer-friendly UI for tracking code changes
- Support for [CI/CD or Terraform]() infrastructure as code — deploy in the way most convenient for you
